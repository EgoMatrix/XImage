## 一、XImage 是什么

XImage 是一个**本地优先的跨平台 AI 生图桌面客户端**。

- **多模型多渠道**：OpenAI、Google、字节、MiniMax、Kimi，以及任何兼容渠道，可在「渠道设置」里配置。
- **本地优先**：API Key、生成历史、图片文件全部保存在你自己的电脑上。
- **结构化创作**：把模糊的「画一张好图」拆成「选模式 → 填字段 → AI 策划 → 生成」四步。
- **免费可用**：所有功能永久免费，作者通过赞助维持长期维护。
- **数据安全**：所有数据都存储在自己电脑上，随时可用。


### 适合谁

- 内容创作者（小红书 / 抖音 / 公众号 / 视频号）
- 电商运营、品牌、设计师
- 学生、独立开发者、Indie Hacker
- 已经有 API Key 但不想用网页版的人
- 全职妈妈，没有太多时间又想快速生成高质量图片

### 不做什么

为了保持 MVP 简单，XImage 首版**不做**以下事情：

- 复杂工作流编排
- 模板市场 / 账号体系 / 强制付费
- 图片编辑器（裁剪、修图、抠图）
- 批量任务调度
- 在线协作

## 二、渠道与三类模型（必读）

XImage 用到三类模型，每类都需要一个「渠道」（也就是 API Key + Base URL）：

| 这是什么 | 干啥用的 | 必填吗 |
|---|---|---|
| **文生文模型** | 帮你把简单想法变成漂亮提示词（点"AI 策划"会用） | ✅ 必须 |
| **文生图模型** | 把提示词画成图（点"生成"会用） | ✅ 必须 |
| **图生图模型** | 上传参考图后，让 AI 在原图基础上改 | ⭕ 可选，不传参考图就不用 |

### 最简单：一家全包（推荐新手）

> **直接买一个 OpenAI 渠道**（官方或兼容 OpenAI 的中转商都行），用同一个 Key 就能搞定全部三类。
>
> - 文生文 → `gpt-5.4/gpt-5.5`
> - 文生图 → `gpt-image-2`
> - 图生图 → `gpt-image-2`（同一个，自带图入能力）
>
> 一个 Key、一个渠道、零配置烦恼。**不知道选啥就选这个**。

### 想省钱或想换花样：可以分开配

三类模型**完全可以来自不同渠道**——XImage 只要求每类至少有一个能用就行。常见组合：

| 你想要 | 文生文 | 文生图 | 图生图 |
|---|---|---|---|
| 最省钱 | `deepseek-chat`（便宜） | 中转商的 SDXL 套餐 | 不配（不传参考图就行） |
| 文字封面好看 | `gpt-5.4/gpt-5.5` | `gpt-image-2`（文字渲染强） | `gpt-image-1` |
| 艺术风格强 | `Gemini-3-Flash-Preview` | `flux-pro` 或 `midjourney` | `flux-redux` |
| 全套谷歌 | `Gemini-3-Flash-Preview` | `Gemini-3.1-Flash-Image-Preview` | `Gemini-3.1-Flash-Image-Preview` |

### 三句话记住

1. **AI 策划帮你写提示词** —— 用便宜的小型对话模型就够（不需要图能力）。
2. **生图是大头** —— 看你重视什么：文字 / 艺术 / 速度 / 价格，挑一个适合的。
3. **图生图先不急** —— 你不上传参考图就用不到，等需要时再配。

## 三、安装与启动

### 3.1 下载

到 [Releases 页面](https://github.com/EgoMatrix/XImage/releases/latest) 下载对应平台的安装包：

| 系统 | 安装包 | 适用机型 |
|---|---|---|
| macOS（Apple Silicon） | `XImage-<version>-mac-arm64.dmg` | M1 / M2 / M3 / M4 系列 |
| macOS（Intel） | `XImage-<version>-mac-x64.dmg` | 2020 及之前的 Intel Mac |
| Windows | `XImage-<version>-win-x64.exe` | Windows 10 / 11 64 位 |
| Linux | `XImage-<version>-linux-x64.AppImage` | 大多数主流发行版 |

### 3.2 macOS 安装

1. 双击下载好的 `.dmg`，把 XImage 拖入「应用程序」。
2. 首次启动会提示「**无法验证开发者**」：
   - **方法一（推荐）**：在「访达 / 应用程序」里**右键 XImage → 打开**，然后在弹窗里再点一次「打开」。
   - **方法二**：前往「系统设置 → 隐私与安全性」，滑到底部能看到「已阻止 XImage」，点「仍要打开」。
3. 之后再启动就不会再弹窗。

> 这是 macOS 对未公证应用的默认行为。XImage 没有上传 API Key 也没有发起任何网络请求（除了你主动触发的生图调用），不必担心。 如果提示包损坏，可以打开终端执行 sudo xattr -rd com.apple.quarantine "/Applications/XImage.app"

### 3.3 Windows 安装

1. 双击 `.exe`，按引导完成安装。
2. SmartScreen 可能提示「**Windows 已保护你的电脑**」：点「**更多信息 → 仍要运行**」即可。
3. 桌面会生成快捷方式。

### 3.4 Linux 安装

```bash
chmod +x XImage-<version>-linux-x64.AppImage
./XImage-<version>-linux-x64.AppImage
```

也可以双击运行（部分发行版需要预先安装 `libfuse2`）。

### 3.5 首次启动会发生什么

- 应用会在用户目录下创建 XImage 数据目录（用于存放数据库、配置、缩略图）。
- **不会**自动联网，所有功能都需要你主动配置渠道之后才会发起网络请求。
- **不会**默认上传任何遥测数据。


## 四、5 分钟跑通第一张图

> 下面按"**OpenAI 一家全包**"走最快路径。其它组合到 [第十章](#十渠道模型与设置) 调整。

### 第 1 步：加一个渠道

打开 XImage → 左侧「**渠道设置**」→「**新增渠道**」：

1. 渠道名称：随便填，例如 `OpenAI`
2. Base URL：`https://api.openai.com/v1`（或中转商提供的 URL）
3. API Key：粘贴你的 Key
4. 点「**测试连接**」→ 看到「连接成功」就成
5. 「**设为默认**」并启用

### 第 2 步：启用三类模型（同一个渠道，复制三份）

进入「**模型管理**」，找到刚才那个渠道下：

| 类型 | 启用这个 | 备注 |
|---|---|---|
| 文生文 | `gpt-5.4/gpt-5.5` | 便宜，AI 策划专用 |
| 文生图 | `gpt-image-2` | 文字渲染好，封面海报合适 |
| 图生图 | `gpt-image-2` | 跟上面同一个模型，再勾一次"图生图"能力即可 |

> ⚠️ API Key 在本地加密保存，**不会**上传 XImage 的任何服务器。

### 第 3 步：生成第一张图

回到「**首页**」：

1. 默认就是「内容封面」模式
2. 选个平台（小红书 / 抖音 / 公众号…）
3. 填「内容主题」「封面标题」
4. （推荐）点「**AI 策划**」让模型帮你优化一下
5. 点「**生成封面**」

等几秒到几十秒，预览区出图。

### 第 4 步：保存

- 预览区点「**放大**」看大图
- 「**生图记录**」批量管理、复用提示词
- 点「**保存**」选本地路径

## 五、主界面导览

XImage 主界面分为四个区域：

| 区域 | 入口 | 用途 |
|---|---|---|
| **首页（创作工作台）** | 左侧导航「首页」 | 选模式、填字段、生成图片 |
| **生图记录** | 左侧导航「生图记录」 | 看历史、搜索、复用、保存 |
| **渠道设置** | 左侧导航「渠道设置」 | 管理 API Key、Base URL、测试连接 |
| **模型管理** | 左侧导航「模型管理」 | 启用 / 禁用 / 新增图片模型 |
| **设置** | 右上角齿轮 | 保存路径、参考图数量上限、调用日志等 |
| **关于** | 左下角作者头像 | 版本号、赞助二维码、鸣谢 |

### 4.1 首页布局

桌面屏宽 ≥1321px 时，首页分为左右两列：

- **左列「生成参数」**：创作模式字段 + 上传参考图 + AI 策划卡 + 比例 / 数量 / 质量 + 生成按钮。
- **右列「输出区」**：上方是预览大图（按你选的比例显示），下方是「更多候选」候选图网格。

屏宽较小时（981 ~ 1320px）自动切换为更紧凑的双列布局；<981px 单列。

## 六、九大创作模式

XImage 把不同的生图诉求拆成 9 个模式，每个模式都有自己的字段集合、AI 策划助手与默认风格。

### 5.1 速查表

| 模式 | 默认比例 | 默认风格 | 一句话场景 |
|---|---|---|---|
| 内容封面 | 3:4 | 简约现代 | 小红书、公众号、抖音等分发平台封面 |
| 人物角色 | 16:9 | 高奢人像摄影 | 设定表、转身图、表情组、头像 |
| 商品素材 | 1:1 | 电商高光 | 电商主图、详情图、种草图 |
| 场景设定 | 16:9 | 电影场景 | 空间、世界观、镜头场景 |
| 海报物料 | 4:5 | 电影海报 | 活动、品牌、课程、节日 |
| 特效素材 | 1:1 | 能量光效 | 光效、粒子、烟雾、HUD |
| 表情包 | 1:1 | Q 版表情 | 聊天表情、贴纸、梗图 |
| 创意概念 | 3:4 | 皮克斯 3D | mini-me、巨人国、双重曝光 |
| 自由创作 | 1:1 | 写实摄影 | 没有模板的灵感实验 |

### 5.2 内容封面

**适用场景**：小红书图文、抖音视频封面、公众号头图、视频号封面、知乎专栏等。

**关键字段**：
- 平台（直接决定推荐比例）
- 内容主题（必填）
- 封面标题 / 副标题
- 内容类型（知识分享、生活记录、好物种草、教程指南…）
- 文字模式（含文字 / 无字）

**好例子**：
- 内容主题：`周末三件能让人变快乐的小事`
- 标题：`快乐有时候很简单`
- 内容类型：`生活记录`
- 平台：`小红书`，比例 `3:4`，风格 `简约现代`

**建议**：标题不要超过 12 个字，画面留白会更好看；副标题写在角落能增加层次。

### 5.3 人物角色

**适用场景**：虚拟主播、品牌主理人、IP 角色、游戏角色设计、个人头像组。

**关键字段**：
- 角色名称、身份（必填）
- 性别、年龄段
- 外貌特征（textarea，可点「✨ 随机好看长相」）
- 服装道具（textarea，可点「✨ 随机好看搭配」）
- 输出类型：角色设定表 / 多角度转身图 / 表情组 / 姿态组 / 头像组

**好例子**：
- 角色名称：`林夏`
- 身份：`独立咖啡馆主理人`
- 外貌：`鹅蛋脸 长直发 戴细框眼镜 微笑`
- 服装：`米色针织衫 牛仔围裙 手持咖啡杯`
- 输出类型：`多角度转身图`，比例 `16:9`

**建议**：身份 + 年龄段填好后会自动给出外貌 / 服装的参考描述，可直接生成；不满意就连点几次「✨ 随机」换组合。

### 5.4 商品素材

**适用场景**：电商主图、详情卖点图、场景种草图、包装展示图、品牌物料。

**关键字段**：
- 商品名称（必填）、品类
- 核心卖点（textarea）
- 使用场景（textarea）
- 输出类型：商品主图 / 详情卖点图 / 场景种草图 / 包装展示图 / 品牌物料图
- 卖点文字（无字 / 带卖点）

**好例子**：
- 商品名称：`便携手冲咖啡套装`
- 品类：`数码`
- 卖点：`一壶一杯一滤纸 30 秒搞定 出差也能喝精品`
- 场景：`高铁桌板上 城市出差 一边办公一边冲一杯`
- 输出类型：`场景种草图`，比例 `1:1`

### 5.5 场景设定

**适用场景**：空间、环境、世界观设定，游戏关卡、电影场景、品牌空间提案。

**关键字段**：
- 场景名称、地点空间（必填）
- 氛围情绪（textarea）
- 关键元素（textarea）
- 镜头视角、时间光线

**好例子**：
- 场景：`未来书店`
- 地点：`未来城市`
- 氛围：`安静 治愈 灯光柔和 一个人静静地看书`
- 元素：`悬浮的发光书 玻璃穹顶 倒映的城市天际线`
- 镜头：`平视中景`，光线 `夜景灯光`

### 5.6 海报物料

**适用场景**：活动宣传、品牌海报、课程招生、节日海报、新品发布、社群招募。

**关键字段**：
- 物料主题（必填）、主标题
- 辅助信息（textarea）
- 目标人群（textarea）
- 物料类型、文字排版（含文字 / 无字底图）

**好例子**：
- 物料主题：`周末手冲咖啡体验课`
- 主标题：`30 分钟学会一杯好咖啡`
- 辅助信息：`周六下午 14:00 三里屯店 限 12 人 提前报名享 8 折`
- 目标人群：`咖啡爱好者 年轻白领 想自己冲咖啡的人`
- 物料类型：`课程招生`，比例 `4:5`

### 5.7 特效素材

**适用场景**：短视频包装、游戏技能、商品合成、海报氛围、直播贴片。

**关键字段**：
- 特效名称、特效类型（光效粒子、能量爆裂、烟雾火焰、科技 HUD…，25+ 类）
- 动态方向（textarea）
- 色彩倾向（textarea）
- 背景方式（透明 / 深色 / 浅色）
- 使用场景（textarea）

**好例子**：
- 特效：`蓝色能量护盾`
- 类型：`能量护盾`
- 动态：`从中心向外扩散 带波纹涟漪`
- 色彩：`蓝紫霓虹 边缘金色高光`
- 背景：`透明背景`，比例 `1:1`

### 5.8 表情包

**适用场景**：聊天表情、贴纸、梗图角色、品牌 IP 一组情绪表情。

**关键字段**：
- 表情主体（textarea，必填）
- 核心情绪（textarea）
- 表情类型：单张 / 四宫格 / 九宫格 / 贴纸套组 / 梗图
- 动作表情（多选）
- 表情文字（textarea）

### 5.9 创意概念

**适用场景**：电影感画面、超现实合成、戏剧化情节、广告大片。

**关键字段**：
- 画面主体（textarea，必填）
- 概念类型：Mini-me、巨人国、真人 + 卡通、双重曝光、漂浮失重、被卷入海报/书本/屏幕、电影分镜动作、超现实物理 …
- 戏剧情节（textarea）
- 副主体 / 镜头玩法（textarea）
- 情绪氛围（textarea）
- 特别要求（textarea）

**好例子**：
- 主体：`一个年轻女孩坐在公园长椅上看书`
- 概念：`巨人国 / 微观世界`
- 情节：`一只猫从书页里走出来 比女孩还大`
- 镜头：`低机位仰视 浅景深`
- 氛围：`梦幻 童话 治愈`

### 5.10 自由创作

**适用场景**：没有固定模板、想直接描述画面的情况。

**关键字段**：
- 主体、创意一句话（任选一个填）
- 视觉手法（textarea）
- 情绪氛围（textarea）
- 特别要求（textarea）

> **小技巧**：如果你在其它八个模式里有了好的画面，但想再「自由发挥一下」，可以复制提示词到「自由创作」并改写。

## 七、AI 策划助手

每个模式的右侧（或字段下方）都有一个「**AI 策划**」按钮。它做的事情是：

1. 把你填的字段 + 平台 / 风格信息组装成一段「原始提示词」。
2. 调用一个**专门的提示词优化模型**，按当前模式的最佳实践重写。
3. 输出「优化后提示词 + 风格 + 构图 + 负面词」，应用到生成请求里。

### 6.1 何时点 AI 策划

- 字段填得比较散、不知道画面会出来啥
- 上一张生成的不够好、想换一个方向
- 想看看 AI 会怎么组织你的想法（学习用）

### 6.2 完整提示词怎么编辑

点击「**完整提示词**」按钮，会弹出可编辑的提示词面板：

- **创作输入**：你在字段里填的东西，只读展示。
- **优化提示词**：AI 输出的，可以手工改。
- **负面提示词**：用于排除不想要的元素。
- 「**重新生成**」会基于你的最新编辑再让 AI 优化一次。
- 「**应用修改**」把当前编辑结果作为下一次生成的提示词。

### 6.3 调用渠道

AI 策划默认使用你设置的「策划渠道」，可以和图片生成是不同渠道（例如用 OpenAI / Gemini 做策划，用 Stable Diffusion 系做生成）。

## 八、参考图：风格借鉴与图生图

在「上传参考图」处可以上传 1 ~ 4 张图（数量上限可在「设置」里调整）。

### 7.1 两种工作模式

XImage 会根据你**有没有上传参考图**自动切换：

- **无参考图**：纯文本生成（generation）。
- **有参考图**：图生图（remix），渠道与模型会自动切换到支持图生图的版本。

### 7.2 多张参考图怎么用

- 1 张：通常作为「主参考」，控制构图或角色。
- 2~4 张：作为「风格/元素借鉴」，模型会综合参考。

### 7.3 参考图小技巧

- **保留角色**：在字段里写「参考图中的写实角色（保留面部特征、脸型、肤色、发型与身份）」，并配「人物角色」模式。
- **风格借鉴但换主体**：参考图传风格图，主体在字段里写新的描述。
- **被卷入海报/书本/屏幕**：「创意概念」模式选 `被卷入海报/书本/屏幕`，参考图传海报或屏幕截图。

## 九、生图记录、保存与复用

### 8.1 生图记录页

进入「**生图记录**」可以：

- 按提示词 / 作品来源 / 模型 / 渠道搜索
- 按时间、模型、渠道、作品来源筛选
- 点击图片放大查看
- 查看「原始提示词 / 优化后提示词 / 模型 / 渠道 / 比例 / 时间 / 保存路径」
- 「**重新生成**」复用提示词
- 「**保存**」到本地任意目录
- 「**打开文件夹**」直接定位到本地文件

### 8.2 候选图

首页右下角的「**更多候选**」会展示本次生成的其它候选，以及历史里同主题的相关图，可以一键放大或「设为参考图」。

### 8.3 数据存哪里

- 生成历史：本地 SQLite 数据库
- 缩略图：本地数据目录
- 大图：保存到你在「设置」里指定的图片目录

具体路径可在「**设置 → 数据存储**」里看到。

## 十、渠道、模型与设置

> 三类模型的概念和组合建议在 [第二章](#二渠道与三类模型必读) 已经讲过，这里只讲**怎么操作**。

### 10.1 渠道页能干嘛

「**渠道设置**」页面里管理一组组 API 入口，每条渠道有：

- 渠道名称（自定义）
- 类型：OpenAI 官方 / 中转 API / 自定义 Base URL
- Base URL、API Key、代理、超时
- 启用状态、是否默认

按钮：
- **新增 / 编辑 / 删除**渠道
- **测试连接** —— 一键确认 Key 有效
- **拉取模型列表** —— 自动列出该渠道支持的模型 ID
- **设为默认** —— 决定首次打开时用哪个

### 10.2 模型页能干嘛

「**模型管理**」页面里启用 / 禁用 / 新增模型。每个模型有：

- 模型 ID（如 `gpt-image-1`、`gpt-4o-mini`）
- 归属渠道
- 能力勾选：**文生文 / 文生图 / 图生图**（可多选）
- 启用状态

按钮：
- **启用 / 禁用** —— 控制是否出现在工作台下拉里
- **新增自定义模型** —— 渠道方支持但 XImage 没预置的模型 ID 自己加
- **能力标记** —— 勾错了工作台会过滤掉，记得对应

### 10.3 工作台怎么自动选模型

- 点「**AI 策划**」 → 只显示有「文生文」能力的模型
- 点「**生成**」且**没传参考图** → 只显示有「文生图」能力的模型
- 点「**生成**」且**传了参考图** → 只显示有「图生图」能力的模型

> ⚠️ 如果某个模型只勾了「文生图」、没勾「图生图」，那你传参考图时它**不会**出现——这是设计如此，避免送图给不支持图入的模型报错。

### 10.4 设置页能干嘛

「**设置**」页面里能调：

- 默认保存路径
- 参考图数量上限（1 ~ 4 张）
- 是否允许多参考图
- 默认渠道
- **调用日志**（出错时来这里看具体原因）
- 数据存储路径

## 十一、常见问题

详细 FAQ 见 [docs/faq.md](faq.md)。下面是最常见的 5 个：

1. **「测试连接」失败怎么办？**
   - 先检查 Base URL 拼写、Key 是否过期、是否需要代理。
   - 在「设置 → 调用日志」看具体错误信息。

2. **生成的图带文字总是糊的？**
   - 文字渲染对模型挑战较大，可在字段里把标题写得**短一点**（≤12 字），并选「**含文字**」模式让模型集中渲染该字。

3. **想用 16:9 但生成出来变形？**
   - 不同模型对比例支持范围不同。GPT-Image 系列对 16:9 支持好；某些模型只支持 1:1 / 3:4 / 4:3，可换模型试试。

4. **API Key 会上传吗？**
   - 不会。XImage 完全本地运行，Key 仅用于直接调用你设置的 Base URL。

5. **数据怎么备份？**
   - 复制「设置 → 数据存储」显示的目录即可，整体迁移到另一台机器也能继续用。
