# docs/images

公开仓 README 与 user-guide.md 引用的截图与图片资源放在这里。

发版前需要补齐：

| 文件 | 用途 | 建议尺寸 |
|---|---|---|
| `banner.png` | README 顶部横幅 | 1280×400 |
| `screenshot-home.png` | 生图工作台 | 1440×900 |
| `screenshot-works.png` | 生图记录 | 1440×900 |
| `screenshot-channels.png` | 渠道与模型管理 | 1440×900 |
| `screenshot-modes.png` | 九大创作模式 | 1440×900 |

注意事项：
- 截图请打码 API Key、Token、本地用户名路径
- 文件名用小写英文+连字符
- 大于 1 MB 的截图请压缩（建议 PNG 优化或 WebP）
